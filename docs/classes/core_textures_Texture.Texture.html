Class Texture

Texture class: Used to create [textures]GPUTexture or [external textures]GPUExternalTexture from different kinds of [sources]TextureSource. Handles the various sources loading and uploading, GPU textures creation, [texture matrix binding]BufferBinding and [texture binding]TextureBinding
