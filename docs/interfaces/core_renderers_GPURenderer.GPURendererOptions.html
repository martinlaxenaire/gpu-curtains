Type declaration

  • OptionaluseDepth?: boolean

    Whether the renderer RenderPass should handle depth. Default to true.

  • OptionalsampleCount?: number

    The renderer RenderPass sample count (i.e. whether it should use multisampled antialiasing). Default to 4.

  • OptionalcolorAttachments?: [Partial<ColorAttachmentParams>, ...ColorAttachmentParams[]]

    Array of one or multiple ColorAttachmentParams. Useful to set a default clear value.
